Research Title: Real Time Monitoring of Animal Feeding Behaviour Using Computer Vision & Embedded Systems

Developed a farm-deployed system combining YOLOv8n, SORT, ultrasonic trigger + ESP32 + load-cells to detect individual pigs (mAP 97%), measure feeding duration and per-visit feed consumption, and upload time-stamped feeding events to the cloud via MQTT/Firebase.

Research Title: A Non-Intrusive Approach To Measuring Flow Rate In A Pipe Using Machine Learning

Designed a non-intrusive system to measure water flow in pipes using accelerometer vibration data and machine learning. Developed and compared models (Decision Tree, Random Forest, DNN, 1-D CNN), with CNN achieving R² = 0.93 for accurate real-time flow prediction.
